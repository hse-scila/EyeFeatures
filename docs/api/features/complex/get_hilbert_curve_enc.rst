get_hilbert_curve_enc
=====================

.. currentmodule:: eyefeatures.features.complex

.. autofunction:: get_hilbert_curve_enc

Check for :ref:`get_hilbert_curve` method. It is the same
method except that resulting values are encoded as a feature vector (boolean encoding),
where each binary value represents presence of fixations in the
corresponding block.
